# Indexing

Indexing is a special operation available with `[` and `]`.

## Arrays

```ruby
example = [1, 2, 3]
println(example[0]) # Outputs: 1
```

## Tuples

```ruby
example = [1, 2, 3]
println(example[0]) # Outputs: 1
```

## Hash tables

```ruby
example = {"john": 32, "alice": 30}
println(example["alice"]) # Outputs: 30
```
